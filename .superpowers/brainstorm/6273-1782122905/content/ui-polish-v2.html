布局优化要点

  • GitHub Dark 配色方案 — 采用 GitHub 官方深色主题色系（#0d1117 背景），视觉更舒适
  • SVG 图标替代 Emoji — 所有分支/标签/远程图标使用 SVG 矢量图标，更清晰专业
  • 标签页圆角药丸样式 — 活跃标签蓝色背景，非活跃灰色，视觉层次分明
  • HEAD 行背景高亮 — 使用 #0d1926 深蓝背景，而非纯色块
  • 状态栏 SVG 图标 — SSH/远程/镜像状态使用对应图标，信息密度更高
  • 搜索框内嵌图标 — 搜索框左侧内嵌放大镜 SVG，交互提示更明确
  • 页面切换按钮 — 活跃页带边框 + 蓝色高亮，非活跃页无边框灰色文字
  • Conventional Commit 彩色前缀 — feat(绿)/fix(红)/docs(蓝)/merge(紫) 自动着色
  • 头像占位符 — 使用 SVG 人物轮廓替代文字"头像"，更美观
  • 进行中操作栏 — 终止按钮使用红色边框危险样式，操作优先级一目了然
