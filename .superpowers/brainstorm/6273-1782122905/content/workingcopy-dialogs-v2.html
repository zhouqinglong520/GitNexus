工作区 + 对话框优化要点

  • 文件状态 SVG 图标 — Modified(绿色箭头) / Added(绿色加号) / Deleted(红色箭头+删除线) / Renamed(黄色双向箭头)，替代纯文字 M/A/D/R
  • 增删统计彩色分离 — +N 绿色 / -N 红色，等宽字体右对齐
  • 分组计数徽标 — "未暂存" 红色圆角 / "已暂存" 蓝色圆角
  • 输入框焦点样式 — 蓝色边框 + 外发光（box-shadow: 0 0 0 3px rgba(88,166,255,0.15)）
  • 按钮系统 — 4 种变体: Primary(蓝) / Secondary(灰) / Success(绿) / Danger(红)，统一 6px 圆角
  • 对话框 — 10px 圆角 + 深色阴影 + 图标标题，危险操作带红色警告图标
  • 通知 Toast — 圆形图标 + 标题 + 描述 + 关闭按钮，支持成功/错误/警告/信息 4 种类型
  • Checkbox 主题色 — accent-color: #58a6ff，与整体蓝色系一致
  • Diff 背景色 — 新增行 #0d1a0d 深绿底 + 绿字，删除行 #1a0d0d 深红底 + 红字
